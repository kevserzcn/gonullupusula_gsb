1.SQL Server Management Studio (SSMS)'yi açın ve SQL Server sunucunuza bağlanın.
2.Bağlantı sağlandıktan sonra, Object Explorer penceresinde Databases klasörüne sağ tıklayın ve Attach... seçeneğini tıklayın.
3.Karşınıza çıkan pencerede Add butonuna tıklayın.
Açılan dosya penceresinde, ilgili .mdf dosyasını bulun ve seçin.
4.Dosyayı seçtikten sonra OK butonuna basarak veritabanını SQL Server'a bağlayın.

//Projede app.py ve ai.py dosyalarında veritabanı yolu belirtilerek veri çekilmiştir. Projeyi açarken yolları kendi bilgisayarınızdaki yola ve sunucuya göre açınız.
